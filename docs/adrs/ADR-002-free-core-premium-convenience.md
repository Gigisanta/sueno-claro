# ADR-002: Free core calculator with premium convenience

## Status
Accepted

## Date
2026-06-30

## Context

RevenueCat data shows hard paywalls can produce much higher RPI, but this product depends on SEO trust and instant utility. Blocking the calculator would undercut acquisition.

## Decision

Keep the core calculator free. Monetize web with gentle ads and native app with optional premium: no ads, saved routines, custom settings, widgets, and reminders.

## Consequences

- Lower immediate conversion than hard paywall.
- Better SEO/shareability/trust.
- Premium must focus on repeat convenience, not access to the basic answer.
