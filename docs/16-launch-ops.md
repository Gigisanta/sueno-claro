# Launch operations

## Prelaunch

- Reserve domain.
- Create simple logo/icon.
- Build web MVP.
- Set up privacy policy and disclaimer.
- Set up Search Console.
- Add basic analytics with privacy-safe events.
- Prepare feedback email/form.

## Soft launch

- Publish web silently.
- Index sitemap.
- Share with 10-20 users manually.
- Observe calculator completion and copy confusion.
- Fix UX before broader launch.

## Public launch channels

1. SEO pages.
2. Reddit communities only where self-promotion rules allow; lead with utility, not hype.
3. Hacker News “Show HN” after polished PWA.
4. Product Hunt only after visual polish and app/PWA install path.
5. Short videos: “If you need to wake at 7:30, sleep at…”
6. Spanish content for LatAm long-tail.

## Operational checklist

- Uptime monitor for homepage.
- Weekly Search Console review.
- Monthly content refresh.
- Quarterly pricing review.
- App Store review prep once iOS starts.

## Support docs needed before paid launch

- Privacy policy.
- Terms/subscription terms.
- Refund instructions.
- Contact/support email.
- Medical disclaimer.
